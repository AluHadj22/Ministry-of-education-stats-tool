Анализатор отчётов Министерства просвещения
Автоматизированная система анализа отчётов федерального центра Министерства просвещения с визуализацией статистических данных.

Обзор
Программа на Python для обработки официальных отчётов Министерства просвещения. Система извлекает ключевые показатели, проводит статистический анализ и формирует наглядные диаграммы и сводные отчёты. Решение помогает оперативно выявлять тенденции, сравнивать показатели по районам и принимать обоснованные управленческие решения.

Поддержка форматов: обработка PDF, XLSX
